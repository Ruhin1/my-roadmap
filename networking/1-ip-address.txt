IP ঠিকানা (IP Address) হলো "Internet Protocol Address" এর সংক্ষিপ্ত রূপ। এটি এমন একটি সংখ্যার ঠিকানা যা ইন্টারনেটে সংযুক্ত প্রতিটি ডিভাইসকে সনাক্ত করার জন্য ব্যবহৃত হয়। এটি ইন্টারনেটের মাধ্যমে ডেটা প্রেরণ এবং গ্রহণের সময় ডিভাইসগুলোর মধ্যে একটি ইউনিক (অদ্বিতীয়) পরিচয় প্রদান করে।

IP ঠিকানার প্রকারভেদ:

IP ঠিকানাকে প্রধানত দুইটি প্রোটোকলের মাধ্যমে ভাগ করা যায়:

১. IPv4 (Internet Protocol Version 4)

এটি ১৯৮০-এর দশকে চালু হয়।

একটি IPv4 ঠিকানায় ৩২-বিট সংখ্যা থাকে।

এটি ডট ডেসিমাল ফর্ম্যাটে লেখা হয়, যেমন: 192.168.0.1।

এর মাধ্যমে মোট ৪.৩ বিলিয়ন (প্রায়) ঠিকানা প্রদান সম্ভব।

উদাহরণ:

ব্যক্তিগত নেটওয়ার্কের জন্য: 192.168.x.x, 10.x.x.x, 172.16.x.x - 172.31.x.x

পাবলিক ঠিকানা: যা ইন্টারনেটে ব্যবহৃত হয়।



২. IPv6 (Internet Protocol Version 6)

IPv4 এর সীমাবদ্ধতা কাটিয়ে উঠতে ১৯৯৮ সালে IPv6 চালু হয়।

এটি ১২৮-বিট সংখ্যা ব্যবহার করে।

একটি IPv6 ঠিকানা হেক্সাডেসিমাল ফর্ম্যাটে লেখা হয়, যেমন: 2001:0db8:85a3:0000:0000:8a2e:0370:7334।

IPv6 অসংখ্য ঠিকানা সরবরাহ করতে পারে, যা ভবিষ্যতের জন্য যথেষ্ট।



---

IP ঠিকানার দুটি ধরন:

১. স্ট্যাটিক (Static IP)

এটি স্থায়ী এবং পরিবর্তন হয় না।

এটি সাধারণত সার্ভার বা গুরুত্বপূর্ণ ডিভাইসে ব্যবহৃত হয়।

উদাহরণ: ওয়েব সার্ভার, মেইল সার্ভার।


২. ডাইনামিক (Dynamic IP)

এটি পরিবর্তনশীল এবং ইন্টারনেট সার্ভিস প্রোভাইডার (ISP) দ্বারা স্বয়ংক্রিয়ভাবে বরাদ্দ করা হয়।

সাধারণ ব্যবহারকারীদের জন্য এটি সাশ্রয়ী।



---

IP ঠিকানার অংশগুলো:

একটি IP ঠিকানায় দুইটি প্রধান অংশ থাকে:

1. নেটওয়ার্ক আইডি (Network ID): এটি নির্দেশ করে কোন নেটওয়ার্কের অন্তর্গত।


2. হোস্ট আইডি (Host ID): এটি নির্দিষ্ট ডিভাইস বা হোস্টকে চিহ্নিত করে।




---

IP ঠিকানা কীভাবে কাজ করে?

1. যখন একটি ডিভাইস ইন্টারনেট ব্যবহার করে, তখন সেটি একটি IP ঠিকানা পায়।


2. IP ঠিকানার মাধ্যমে ডেটা প্যাকেট ডিভাইস থেকে নির্দিষ্ট গন্তব্যে পৌঁছে।


3. রাউটার এবং DNS সার্ভার IP ঠিকানার সাহায্যে ডেটা প্রেরণ এবং গ্রহণে সহায়তা করে।




---

গুরুত্বপূর্ণ বিষয়:

IP ঠিকানা ডোমেইন নামের সাথে যুক্ত হতে পারে (যেমন, www.example.com এর IP ঠিকানা হতে পারে 93.184.216.34)।

ডোমেইন নাম সনাক্ত করার জন্য DNS (Domain Name System) ব্যবহৃত হয়।


আপনার আরও কোনো প্রশ্ন থাকলে জানাবেন।
